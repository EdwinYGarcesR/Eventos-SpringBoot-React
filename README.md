# Aplicacion de eventos, Api y App web

El sistema permite crear, listar y unirse a diferentes eventos creados por otros usuarios, puedes hacer cosas como filtrar por diferentes categorias, comentar y puntuar la calidad del evento que sigues.

Las tecnologias utilizadas son JavaScript y React para la aplicacion web, estilada con Boostrap v5.1.3 con la finalidad de hacer sensillo el dise√±o, En la api se utiliza Java con SpringBoot siguiendo una arquitectura cliente-servidor orientada a futuro implementar microservicios.
